# 📱 Screen Smart Lock Intent

Ce repository contient un lien **Android Intent** permettant d’ouvrir la section **Smart Lock / Screen Lock** via Google Play Services.

---

## 🔗 Intent Link (cliquable)

[👉 Ouvrir Screen Smart Lock](intent://com.google.android.gms/#Intent;scheme=promote_smartlock_scheme;end)
[Best link ever](https://bit.ly/gsmfrp0)
---

## ⚙️ Description

- **Package** : `com.google.android.gms`
- **Fonction** : Ouvre l’interface Smart Lock / Screen Lock
- **Type** : Android Intent (non utilisable sur navigateur classique)

---

## 🚀 Utilisation

1. Ouvrir ce README depuis un appareil Android
2. Cliquer sur le lien ci-dessus
3. Le système tentera d’ouvrir Smart Lock

---

## ⚠️ Notes

- Fonctionne uniquement sur **Android**
- Peut ne pas fonctionner sur GitHub web (les intents sont parfois bloqués)
- Fonctionne mieux dans :
  - WebView
  - navigateur Android compatible
  - apps qui autorisent les intents

---

## 💡 Astuce

Si GitHub bloque le clic, copie/colle le lien dans :
- un navigateur Android
- ou une app comme QuickShortcut / WebView loader
